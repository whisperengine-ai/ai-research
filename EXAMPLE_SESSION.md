## How to Run This Yourself

```bash
# Install dependencies
pip install -r requirements.txt
python -m spacy download en_core_web_md

# Setup OpenRouter API (or use local models)
cp .env.example .env
# Edit .env and add OPENROUTER_API_KEY

# Run the chatbot (auto-detects GPU/CPU/Metal)
python consciousness_chatbot.py

# Run with local model and heuristic mode for fast testing
python consciousness_chatbot.py --heuristic
```

**Device Detection happens automatically:**
- **macOS**: Metal GPU (MPS) if available
- **Linux/Windows**: NVIDIA CUDA if available
- **Fallback**: CPU for any platform
